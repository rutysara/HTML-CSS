Learning how to play audio in HTML5.

  2. This one is more compatible and has the same principle as above.
    You add other audio sources types together like OGG and WAV (I do not have other types so only mp3 was used). Put audio types by order that you want the browser to load first.
    In case no audio is compatible with the browser you can add a paragraph at the end but inside the audio tags to load.

Both of these options have very similar attributes, like controls and autoplay, however, it's important to note that many modern web browsers have restrictions on autoplaying audio, especially when it's not initiated by user interaction.

Note: WAV is top quality but very heavy and may slow your website. Use it for download only if at all.
