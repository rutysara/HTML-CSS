Tagging images in HTML5

Choose an image and save it into the same folder as your website.
Then, type img in the program and the image tag will appear.

We can also load images in sub-folders - it must still be inside the main webpage folder!
In this case, just create a sub-folder "images" for example and add to src. It will be represented as src="images/something" and on and on.
You get the idea.

To load images from external sources it is similar. You just need to copy the image address.

However, if the external source is altered or deleted I suppose it will affect my website too. So best to have it downloaded.
