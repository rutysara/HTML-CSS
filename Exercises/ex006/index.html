Ordered Lists

HTML element used to create numbered lists. It organizes items in a sequential order, such as steps in a procedure or a ranked list. Each list item is indicated with a number.

Ordered lists are important because they help you provide instructions or information in a clear step-by-step order.

For example: When you write down a recipe, using numbers to list the steps ensures that someone can follow the recipe correctly. Without numbers or order, it would be confusing, and the end result might not be what you expect. Ordered lists keep things organized and easy to follow.
Another thing is ordered lists automatically number each item, keeping things clear. Adding something in between is easy; you don't have to renumber everything.

  1. To create a list type OL (Ordered List)
  2. To list items, type LI
  3. On HTML5 you do not need to close this tag (optional)
  4. And on and on!
  5. To personalise numbering, simply use TYPE= inside opening OL tag.

Unordered Lists

Unordered lists are a way to present information in a bulleted format, without any specific order or sequence.

They are used when the order of items doesn't matter, and you want to list them without any numerical or alphabetical indication. Each item in an unordered list is preceded by a bullet point or another marker, making it easy to distinguish individual items. Unordered lists are created using the `UL` HTML element.

  • To create a list type UL (Ordered List)
  • To list items, type LI the same way as ordered lists.
  • And on and on!
  • To personalise numbering, simply use TYPE= inside opening UL tag.
