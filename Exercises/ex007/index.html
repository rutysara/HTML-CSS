External Links

External Links: hyperlinks that point to web pages or resources located outside the current website. They are created using the `A` (anchor) element and have an `HREF` attribute specifying the URL of the external page or resource.

Example: My GitHub Repository

Targets: The `target` attribute in HTML specifies where the linked content should open when clicked. Common values include `_blank` (opens in a new browser tab or window) and `_self` (opens in the same tab or window).

  • _blank: `Visit Example`
  • _self: `Visit Example`

Rel Attribute: Using "next," "prev," or "external" values in the "rel" attribute of your links helps web browsers and search engines understand the relationship between different pages on your website, improving navigation and SEO:

  • "next": Indicates the next page in a sequence, helping users and search engines follow a logical order, such as in paginated content.
  • "prev": Indicates the previous page in a sequence, aiding in backward navigation.
  • "external": Signals that a link leads to an external website, allowing the browser to handle it appropriately and providing clarity to search engines.
  • "nofollow": attribute used in HTML to tell search engines not to follow a particular link. It's often used when you don't want to pass on your website's SEO "credit" or authority to the linked page. It's like saying, I'm linking to this, but I don't vouch for it.
These values enhance the user experience and search engine optimization by providing context and structure to your website's content and links.

Internal Links

Internal links are used to connect different pages within the same website or domain. They help users navigate between related content, making it easier to find information and explore a website's structure.

Examples:

  • Page 2
  • Page 3
